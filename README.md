# NEXUS // MODULE SUITE

Nexus is a behavioral utility suite for automated typing and game interaction. It mimics human behavior to avoid detection through randomized input patterns.

## KEY FEATURES
- **Human Typing:** Simulates fatigue, variable pauses, and realistic typos.
- **Focus Control:** Automatically pauses typing if the target window loses focus.
- **Universal Bootstrap:** Detects and installs required Python libraries on first launch.
- **System Tray Integration:** Runs in the background with a quick-access tray icon.

## CONTROLS
- **Start Typing:** `CTRL + ALT + V`
- **Hide/Show UI:** `ALT + C`
- **Emergency Stop:** `ESC` (Kills the typing engine immediately)

## SETUP & REQUIREMENTS
1. Requires Python 3.8 or higher.
2. Download **NEXUS.pyw** to your local machine.
3. Run the file. *Note: The first launch may take a moment as it installs dependencies (Pillow, PyAutoGUI, Pynput, etc.).*
4. Ensure your clipboard contains the text you wish to automate.
